{{- /* Build a once-cached map of rule id -> published rule page permalink. Consumed by the render-codespan hook so any `MDSxxx` inline-code token auto-links to its rule page with zero author markup. Keyed off `.Params.id` (e.g. "MDS001") set in each rule README's front matter. The map is empty when a docs tree is built without internal/rules/index.md (no /docs/rules/ section): the caller then renders a plain span. */ -}} {{- $m := dict -}} {{- with site.GetPage "/docs/rules" -}} {{- range .Pages -}} {{- $id := .Params.id -}} {{- if $id -}} {{- $m = merge $m (dict $id .RelPermalink) -}} {{- end -}} {{- end -}} {{- end -}} {{- return $m -}}
